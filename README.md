## English

### Description
Discover **© ZzzZ - The Never-Ending Puzzle**. A premium, minimalist alphabet-merging puzzle game designed for Android. 
Free, ad-free, and open source.

Instead of numbers, you merge the alphabet. Start with A, merge it with another A to get B, and keep climbing to Z... and way beyond! The game doesn't end at Z; discover combinations like AA, AB, and keep challenging your mind on a board that never stops.

### Key Features
*   **Infinite Gameplay:** No limits. How far can you go?
*   **Addictive Mechanics:** Swipe to move all tiles. Matching letters evolve instantly.
*   **Premium Design:** Clean, modern, and minimalist interface with smooth animations.
*   **Full Customization:** Multiple color themes and Dark/Light mode support.
*   **Offline:** Play wherever and whenever you want.

### How to Play
1.  **Swipe** your finger across the screen to move all tiles.
2.  When two tiles with the same letter touch, they **merge** into the next letter! (A+A = B, B+B = C...).
3.  Keep the board clear and don't let it fill up!

## Español

### Descripción
Descubre **© ZzzZ - The Never-Ending Puzzle**. Un juego de lógica premium y minimalista de fusión de letras para Android. 
¡Gratis, sin publicidad y de código abierto!

En lugar de números, fusionarás el alfabeto. Empieza con la A, únea con otra A para obtener la B, y sigue escalando hasta la Z... ¡y mucho más allá! El juego no termina en la Z; descubre combinaciones como AA, AB, y sigue desafiando a tu mente en un tablero que nunca se detiene.

### Características Principales
*   **Juego Infinito:** Sin límites. Pon a prueba tu estrategia y paciencia.
*   **Mecánica Adictiva:** Desliza para mover las fichas. Las letras iguales se fusionan y evolucionan al instante.
*   **Diseño Premium:** Interfaz limpia y moderna con animaciones fluidas.
*   **Personalización Total:** Múltiples temas de colores y soporte para Modo Claro/Oscuro.
*   **Offline:** Juega donde quieras sin necesidad de conexión.

### Cómo Jugar
1.  **Desliza** el dedo para mover todas las fichas.
2.  Cuando dos fichas con la misma letra se tocan, **¡se fusionan!** (A+A = B, B+B = C...).
3.  Mantén el tablero despejado y evita que se llene.
